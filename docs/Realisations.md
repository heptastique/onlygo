# Réalisations

## `POST /realisation/add` :key:
Ajoute une réalisation et l'associe à la prochaine activité du programme.

### Paramètres
```json
{
	"distance": 2000,
	"date": "2018-04-24T16:04:12",
	"ciId": 10001
}
```

```
Status: 200 OK
```
